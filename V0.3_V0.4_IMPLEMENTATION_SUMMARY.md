## v0.3.0: API Cleanup & Custom Sync Strategies

### Breaking Changes Implemented

✅ **Removed deprecated `onLog` callback** from SyncEngine
- Clean constructor signature
- Forces migration to structured Logger interface
- Reduces legacy API surface

### New Features

✅ **Custom Sync Strategy System**
- Abstract `SyncOrchestrationStrategy` class for extensibility
- `SyncStrategyContext` providing controlled sync access
- 5 built-in strategies:
  1. **StandardSyncStrategy** - Default pull-push pattern
  2. **OfflineFirstSyncStrategy** - Graceful network degradation
  3. **ManualSyncStrategy** - Explicit error handling
  4. **PrioritySyncStrategy** - Table-priority based sync
  5. **CompositeSyncStrategy** - Sequential strategy composition

**New Files:**

  - SyncOrchestrationStrategy abstract base

**Modified Files:**

## v0.4.0: UI Widgets & SyncManager Enhancement

### New Features

✅ **6 Flutter UI Widgets**
- Seamless sync status display without boilerplate
- Pre-styled with Material 3 design language
- Fully customizable via builders and parameters

**Implemented Widgets:**

1. **SyncStatusWidget** (68 lines)
   - Displays sync status with optional progress spinner
   - Manual sync button
   - Custom builder support
   - Status streaming integration

2. **SyncMetricsCard** (90 lines)
   - Card showing pulled/pushed/duration/errors
   - Easy copy-paste integration
   - Configurable elevation and background
   - Hides when empty

3. **SyncErrorBanner** (102 lines)
   - Context-aware error styling
   - Categorizes errors (network, auth, schema, generic)
   - Retry and dismiss buttons
   - Custom message override

4. **OfflineIndicator** (56 lines)
   - Chip showing offline status
   - Only visible when offline
   - Customizable icon and label
   - Custom builder option

5. **SyncButton** (57 lines)
   - Filled button with auto loading state
   - Disables during sync
   - Custom loading indicator support
   - Icon customization

6. **SyncStatusPanel** (142 lines)
   - Comprehensive dashboard combining all widgets
   - Status + metrics + error + button
   - Per-widget visibility control
   - Perfect for settings/dashboard screens

**File Created:**
- `lib/src/ui/sync_widgets.dart` (538 lines)
  - All 6 widget implementations
  - Helper class `_MetricsRow` for consistent styling
  - Comprehensive documentation
  - Usage examples in docstrings
