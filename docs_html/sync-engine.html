Sync Engine

Core orchestrator for all sync operations

The SyncEngine is the central class of syncitron. It orchestrates pulling, pushing, conflict resolution, metrics, and status updates.

Constructor

SyncEngine({
  required LocalStore localStore,
  required RemoteAdapter remoteAdapter,
  syncitronConfig? config,            // default: syncitronConfig()
  Logger? logger,                     // default: ConsoleLogger()
  MetricsCollector? metricsCollector,  // default: InMemoryMetricsCollector()
})

Public API

MethodReturnsDescription
init()Future<void>Ensures sync columns exist. Idempotent — safe to call repeatedly.
registerTable(TableConfig)SyncEngineRegisters a table. Returns this for chaining. Must be called before init().
syncAll()Future<SyncSessionMetrics>Pull + push all registered tables. Skips if sync is already running.
syncTable(TableConfig)Future<SyncMetrics>Sync a single table by config. Skips if sync is already running.
syncTableByName(String)Future<SyncMetrics>Sync a single table by name. Looks up the registered TableConfig.
syncWithOrchestration(strategy)Future<SyncSessionMetrics>Execute a custom SyncOrchestrationStrategy.
dispose()voidCloses the status stream controller. Call on app shutdown.

statusStream

A broadcast Stream<String> emitting human-readable status messages during sync. Wire it to SyncStatusWidget or listen directly:

engine.statusStream.listen((msg) => print('Sync: $msg'));
// "Starting Full Sync..."
// "Downloading todos..."
// "Uploading todos..."
// "Sync completed successfully."

Overlap Protection

If syncAll() or syncTable() is called while a sync is already in progress, the call returns immediately with empty metrics and a warning is logged. This prevents concurrent writes to the local database.

Error Tolerance

syncAll() does not abort on a single table failure. If table A fails, table B is still synced. Errors are captured in SyncSessionMetrics.totalErrors.

final metrics = await engine.syncAll();
if (!metrics.overallSuccess) {
  for (final tm in metrics.tableMetrics) {
    if (!tm.success) print('Failed: ${tm.tableName} — ${tm.errorMessages}');
  }
}
