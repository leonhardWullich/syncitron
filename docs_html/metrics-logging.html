Metrics & Logging

Track performance and debug sync flows

Sync Metrics

Every syncAll() returns a SyncSessionMetrics containing per-table SyncMetrics.

SyncSessionMetrics

PropertyTypeDescription
totalDurationDurationTotal wall time for the session
totalTablesSyncedintNumber of tables processed
totalRecordsPulledintSum of all pulled records
totalRecordsPushedintSum of all pushed records
totalConflictsintSum of conflicts encountered
totalErrorsintNumber of failed table syncs
overallSuccessbooltrue if all tables synced without errors
tableMetricsList<SyncMetrics>Per-table breakdown

SyncMetrics (per table)

PropertyType
tableNameString
durationDuration
recordsPulledint
recordsPushedint
recordsWithConflictsint
conflictsResolvedint
errorsint
successbool

Usage

final metrics = await engine.syncAll();

print(metrics);                        // pretty-printed summary
print(metrics.toJson());               // structured JSON

for (final tm in metrics.tableMetrics) {
  if (!tm.success) {
    print('${tm.tableName} failed: ${tm.errorMessages}');
  }
}

MetricsCollector

ImplementationBehaviour
InMemoryMetricsCollectorStores all sessions and table metrics in memory. Useful for dashboards.
NoOpMetricsCollectorDiscards all metrics. Zero overhead for production.

Logging

syncitron provides a pluggable Logger interface with three built-in implementations.

Logger Implementations

ClassBehaviour
ConsoleLoggerPrints to stdout. Accepts minLevel to filter.
NoOpLoggerDiscards all log output. Zero overhead.
MultiLoggerFan-out: delegates to a list of loggers.

Log Levels

LevelSeverity
LogLevel.debug0 — Verbose debugging
LogLevel.info1 — Normal operations
LogLevel.warning2 — Potential issues
LogLevel.error3 — Errors requiring attention
LogLevel.critical4 — Fatal errors

Custom Logger Integration

class SentryLogger implements Logger {
  @override
  void error(String msg, {Object? error, StackTrace? stackTrace, Map<String, dynamic>? context}) {
    Sentry.captureException(error, stackTrace: stackTrace);
  }
  // ... implement other methods
}

final engine = SyncEngine(
  localStore: store,
  remoteAdapter: adapter,
  logger: MultiLogger([ConsoleLogger(), SentryLogger()]),
);

LogEntry

Structured log entries can be serialized to JSON for export:

final entry = LogEntry(
  level: LogLevel.error,
  message: 'Sync failed',
  error: exception,
  context: {'table': 'todos', 'records': 42},
);
print(entry.toJson());
