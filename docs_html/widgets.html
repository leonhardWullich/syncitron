UI Widgets

6 ready-made Flutter widgets for sync UX

syncitron ships 6 ready-made Flutter widgets for displaying sync status, errors, and metrics.

SyncStatusWidget

Displays sync status from the engine's statusStream.

SyncStatusWidget(
  statusStream: engine.statusStream,
  onSync: () => engine.syncAll(),
  showProgress: true,
  builder: (context, status) => Text(status),  // optional custom builder
)
PropTypeDefaultDescription
statusStreamStream<String>requiredStatus messages from engine
onSyncVoidCallbackrequiredSync trigger callback
builderWidget Function(…)?nullCustom builder, overrides default UI
showProgressbooltrueShow spinner during sync
textColorColor?nullOverride text color

SyncMetricsCard

Shows sync performance metrics in a card layout.

SyncMetricsCard(
  metrics: sessionMetrics,
  elevation: 2,
  backgroundColor: Colors.white,
)
PropTypeDefault
metricsSyncSessionMetricsrequired
elevationdouble1
backgroundColorColor?null
showWhenEmptybooltrue

SyncErrorBanner

Context-aware error banner using Dart 3 pattern matching on the sealed syncitronException hierarchy.

SyncErrorBanner(
  error: currentError,   // syncitronException?
  onRetry: () => engine.syncAll(),
  onDismiss: () => setState(() => currentError = null),
  customMessage: null,   // optional override text
)
PropTypeDefault
errorsyncitronException?required
onRetryVoidCallback?null
onDismissVoidCallback?null
customMessageString?null

When error is null, the banner is hidden (renders SizedBox.shrink()).

OfflineIndicator

Small chip showing online/offline state.

OfflineIndicator(
  isOnline: connectivityStatus,
  offlineIcon: Icons.cloud_off,
  onlineIcon: Icons.cloud_done,
  offlineLabel: 'Offline',
)

SyncButton

A FilledButton.icon that auto-disables and shows a spinner while syncing.

SyncButton(
  onPressed: () => engine.syncAll(),
  isSyncing: isSyncInProgress,
  label: 'Sync Now',
  icon: Icons.sync,
)
PropTypeDefault
onPressedVoidCallbackrequired
isSyncingboolrequired
labelString'Sync'
iconIconDataIcons.sync

SyncStatusPanel

All-in-one dashboard combining status, metrics, errors, and sync button.

SyncStatusPanel(
  statusStream: engine.statusStream,
  onSync: () => engine.syncAll(),
  metrics: lastSessionMetrics,
  error: currentError,
  onErrorDismiss: () => setState(() => currentError = null),
  showMetrics: true,
  showButton: true,
  showStatus: true,
)
PropTypeDefault
statusStreamStream<String>required
onSyncVoidCallbackrequired
metricsSyncSessionMetrics?null
errorsyncitronException?null
onErrorDismissVoidCallback?null
showMetricsbooltrue
showButtonbooltrue
showStatusbooltrue
