❓ FAQ

Q: Do I need to change my code? A: No! Batching is automatic. But updating ReplicoreConfig is recommended.

Q: Will this break anything? A: No. Fully backward compatible.

Q: Which backends benefit most? A: All backends benefit! Firebase, Supabase, Appwrite, GraphQL all optimized.

Q: How is this so fast? A: Instead of 100 individual API calls, it groups them into 4 batch requests.

Q: Can I control batch size? A: Yes! See configuration section above.

Q: What if my backend doesn't support batching? A: Replicore falls back to individual operations automatically.
